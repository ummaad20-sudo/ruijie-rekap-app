# Ruijie Rekap Pro

Aplikasi untuk merekap data dari file Excel Ruijie.

## Fitur
- Pilih file Excel
- Rekap berdasarkan tanggal dan grup
- Copy hasil ke clipboard
- Format Rupiah otomatis

## Cara Install
Download APK dari tab Actions di GitHub.

## Cara Penggunaan
1. Buka aplikasi
2. Klik "PILIH FILE EXCEL"
3. Pilih file Excel dengan format yang sesuai
4. Hasil rekap akan muncul
5. Klik "COPY HASIL" untuk menyalin ke clipboard

## Format Excel yang Didukung
File harus memiliki kolom:
- Grup pengguna
- Harga
